Focus: Nominativ & Akkusativ • B1
🔄 Next Topic

📌 Nominativ (Subject)

The subject of the sentence — the person/thing doing the action.
Der Mann liest. (The man reads.)

📌 Akkusativ (Direct Object)

The direct object — the person/thing receiving the action.
Ich sehe den Mann. (I see the man.)

Fill in the correct articles:
1. Ich sehe den Hund. (👁️ der Hund)
✓ Correct ✗ der
2. Der Lehrer erklärt die Regel. (👨‍🏫 nom.)
✓ Correct
3. Sie kauft den Tisch. (🪑 der Tisch → akk.)
✓ Correct
4. Die Katze schläft auf dem Sofa.
✓ Correct
